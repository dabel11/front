# Week1 WIL (Weekly I Learned)
## 이번 주에 배운 것
- HTML(HyperText Markup Language)의 역사와 기본 구조  
- JS(JavaScript)의 역사와 기초적인 사용법  

## HTML
- 팀 버너스리가 제안한 웹 문서용 언어  
- 제목(h1~h6), 단락(p), 링크(a) 태그로 문서 구조 표현  
- 기본 시작 파일은 index.html 사용  

## 실습 내용
- hello.html을 작성하고 브라우저로 실행 
- index.html이 기본적으로 모든 사이트의 시작페이지로 보통 생략되어 있음
- Live Server를 활용해 코드 변경 실시간 확인
- 짱구 자기소개 페이지와 신형만 자기소개 페이지를 만들며 태그 활용 연습  
- 신형만 페이지를 만들며 hypertext의 기본골자인 링크를 활용하여 페이지를 이동해 보았음

## JavaScript
- 1995년 넷스케이프에서 개발된 스크립트 언어로 웹 문서를 동적으로 조작하기 위해서 만들어짐
- script 태그로 삽입, console.log와 alert로 JS 맛보기
